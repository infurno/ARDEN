# Notes Location Configuration

## Default Notes Directory

```
~/Notes/
```

This is where all notes created by ARDEN will be saved (your second brain).

## Directory Structure

```
~/Notes/
├── 2026-01-02-meeting-with-sarah.md
├── 2026-01-02-project-idea.md
├── 2026-01-03-quick-note.md
└── ...
```

Notes are organized chronologically by filename prefix.

## Current Setting

**Location:** `~/Notes/`

**Auto-create:** Yes (directory will be created if it doesn't exist)

**Permissions:** Read/Write (notes are created with standard file permissions)

## Works great with

- Obsidian
- Logseq
- Notion (markdown export)
- Any markdown editor
- Git version control
