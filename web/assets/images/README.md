# ARDEN Logo

Place your ARDEN logo file here as:
- **arden-logo.png** (main logo)

The logo should be:
- Square or circular format (the circuit board eye design)
- Transparent background recommended
- Reasonable size (200x200 to 512x512 pixels)
- PNG format for best quality with transparency

The logo is already configured to display in:
- Notes page header
- Dashboard header
- Other pages

If the logo file is not found, the image will be hidden automatically.
