## Deploy Now (On Your Server)

You're already on the server, so let's finish the deployment:

### 1. Pull Latest Changes

```bash
cd ~/ARDEN
git pull origin remote-server
```

### 2. Install Node Dependencies

```bash
cd ~/ARDEN/api
npm install
```

This will install the new `express` dependency for the web server.

### 3. Configure .env

Make sure your `.env` has:

```bash
# Telegram
TELEGRAM_BOT_TOKEN=your_bot_token_here

# OpenAI
AI_PROVIDER=openai
OPENAI_API_KEY=sk-your-key-here
OPENAI_MODEL=gpt-4o-mini
USE_OPENAI_WHISPER=true

# Web Server (optional, defaults shown)
WEB_PORT=3000
WEB_HOST=0.0.0.0

# Security
ARDEN_API_TOKEN=<auto-generated>
```

### 4. Start Both Services

```bash
cd ~/ARDEN

# Remove old broken instances
pm2 delete all

# Start fresh with both services
pm2 start ecosystem.config.js

# Check status
pm2 status
```

You should see:
- `arden-bot` - running
- `arden-web` - running

### 5. View Logs

```bash
# Watch both services
pm2 logs

# Or individually
pm2 logs arden-bot
pm2 logs arden-web
```

### 6. Test Telegram Bot

Send a message to your bot on Telegram - it should respond!

### 7. Access Web Dashboard

**Option A: Direct Access (Port 3000)**

```bash
# Allow port 3000
sudo ufw allow 3000
```

Visit: http://rocket.id10t.social:3000

**Option B: Setup NGINX (Recommended)**

See: `docs/NGINX_SETUP.md`

Then visit: http://rocket.id10t.social (port 80)

### 8. Auto-Start on Boot

```bash
pm2 save
pm2 startup
# Follow the command it gives you (run with sudo)
```

## Ports

- **Port 3000**: Web dashboard (internal)
- **Port 80**: NGINX (if installed) → proxies to port 3000
- **Port 443**: NGINX HTTPS (if SSL configured)
- **No port needed**: Telegram bot uses outbound connections only

## Troubleshooting

**Bot not responding:**
```bash
pm2 logs arden-bot
# Check for errors, verify .env has correct tokens
```

**Web dashboard not loading:**
```bash
pm2 logs arden-web
sudo ufw status  # Make sure port 3000 is allowed
curl http://localhost:3000/api/health
```

**Out of space during npm install:**
```bash
df -h
sudo apt clean
sudo apt autoremove
rm -rf ~/.cache
```
